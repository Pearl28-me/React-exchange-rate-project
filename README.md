# Currency Converter

An online platform to convert various currencies to and from Ghana Cedis.

# Features

- User authentication (login, logout, password reset)
- Currency conversion
- Responsive design

# Technologies

- React.js
- Redux
- Sass
- Styled-components
- Exchange Rate API
